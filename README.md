# 📱 Rick and Morty App

Aplicação desenvolvida em **React Native** para consumir a API pública do Rick and Morty e exibir uma lista de personagens de forma dinâmica.

## 🚀 Tecnologias

- React Native
- JavaScript
- Axios
- API REST

## ✨ Funcionalidades

- Listagem de personagens
- Exibição de imagem, nome, espécie e status
- Consumo de API utilizando Axios
- Interface responsiva para dispositivos móveis

## ▶️ Como executar

```bash
git clone https://github.com/Eduardo10101/S1_R4_AT1_PPDM.git

cd S1_R4_AT1_PPDM

npm install

npx expo start
```

## 🌐 API utilizada

https://rickandmortyapi.com/api/character
